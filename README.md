.# Ryan Sports Club Website

A professional, responsive, and dynamic sports club website built using modern front-end web technologies. This project showcases the facilities, annual events, memberships, and gallery of the **Ryan Sports Club**, serving as a digital beacon of athleticism and excellence.

 Features

- **Fully Responsive Navigation:** Built with Bootstrap, featuring a collapse menu for mobile devices and dropdowns for easy facility exploration.
- **Hero Video Background:** An immersive, autoplaying video header that instantly grabs user attention and sets an athletic tone.
- **Dynamic Content Carousel:** Displays different sports categories—**Indoor, Outdoor, and Recreation Sports**—with rich descriptions and links to dedicated sections.
- **Interactive Image Hover Effects:** Custom JavaScript implementation on activity cards that switches images to trophies when hovered, adding an engaging user experience.
- **Comprehensive Sections:** Includes structured landing areas for:
  - Facilities (Indoor, Outdoor, Recreation)
  - Online Membership Form
  - Feedback & News
  - Photo Gallery
  - Contact Us & Head Office location details
- **Social Media Integration:** Connected with quick links to major social platforms (Facebook, Instagram, WhatsApp, Twitter).

 Technologies Used

- **HTML5:** Semantic markup structure.
- **CSS3:** Custom layouts, spacing, and tailored styling (`style.css`).
- **Bootstrap 5:** Fluid grid system, responsive design utilities, navigation bar, and interactive carousels.
- **JavaScript (Vanilla):** Dynamic DOM manipulation for interactive card hovering and visual feedback.
- **Boxicons:** Premium web icons for cleaner UI elements.
